# Futuros Portal

Astro static site for stakeholder presentation.

## Dev

```bash
cd portal
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Deploy to Vercel, Netlify, or any static host.

## Pages

- `/` — landing with thesis + KPIs + roadmap
- `/blueprint` — blueprint structure overview
- `/pilotos` — 5 candidate pilots
- `/paises` — country roles + HPC hubs
- `/kpis` — outcome KPIs + binding inclusion thresholds
- `/preguntas` — 8 open questions

## Style

Dark theme, serif headings (Georgia), sans body, accent color terracotta. Mobile-friendly. Spanish primary.
